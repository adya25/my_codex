# AI Workflow Automation Assistant

Maps repeatable business workflows and recommends automation opportunities to reduce manual effort.

## Project Purpose
This repository is structured as a portfolio-ready AI product prototype. It includes a clean front-end demo, sample data, and Codex-ready guidance so the project can be extended with real integrations later.

## Features
- Workflow intake
- Automation recommendations
- Time savings estimate
- Priority scoring
- Implementation checklist

## Tech Stack
- React
- TypeScript
- Vite
- Rules-based AI mock logic

## Run Locally
```bash
npm install
npm run dev
```
