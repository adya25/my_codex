# Codex Repository Instructions

## Project Context
This is a portfolio-ready AI product prototype for **AI Workflow Automation Assistant**.

## Development Guidelines
- Keep the UI professional, clean, and suitable for a senior AI product portfolio.
- Use accessible labels, readable spacing, and responsive layouts.
- Prefer simple TypeScript types and clear component names.
- Do not add real API keys, credentials, or private customer data.
- Use mock data unless the user explicitly adds a real integration.

## Review Guidelines
- Flag broken UI states, missing empty states, unclear data labels, and hardcoded sensitive values.
- Check that portfolio copy sounds professional and human, not overly generic.
- Make sure screenshots and README content stay aligned with product functionality.
