# AI Customer Insights Dashboard

Analyzes customer feedback, support themes, and behavioral signals to surface actionable product insights.

## Project Purpose
This repository is structured as a portfolio-ready AI product prototype. It includes a clean front-end demo, sample data, and Codex-ready guidance so the project can be extended with real integrations later.

## Features
- Sentiment summaries
- Theme clustering
- Trend snapshots
- Customer priority scoring
- Executive insight cards

## Tech Stack
- React
- TypeScript
- Vite
- Recharts
- Mock feedback data

## Run Locally
```bash
npm install
npm run dev
```
