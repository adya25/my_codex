# AI Document Intelligence Tool

Extracts, summarizes, and organizes key information from business documents for faster review and decision-making.

## Project Purpose
This repository is structured as a portfolio-ready AI product prototype. It includes a clean front-end demo, sample data, and Codex-ready guidance so the project can be extended with real integrations later.

## Features
- Document summary
- Key term extraction
- Action item detection
- Risk flagging
- Searchable document insights

## Tech Stack
- React
- TypeScript
- Vite
- Mock document processor

## Run Locally
```bash
npm install
npm run dev
```
