# AI Automated Jira Project Report

Reviews completed Jira user stories daily, estimates project completion, summarizes status updates, and highlights remaining work and risks.

## Project Purpose
This repository is structured as a portfolio-ready AI product prototype. It includes a clean front-end demo, sample data, and Codex-ready guidance so the project can be extended with real integrations later.

## Features
- Daily story completion review
- Completion forecasting
- Status summaries for stakeholders
- Risk and blocker detection
- Export-ready project report

## Tech Stack
- React
- TypeScript
- Vite
- Recharts
- Mock Jira JSON data

## Run Locally
```bash
npm install
npm run dev
```
